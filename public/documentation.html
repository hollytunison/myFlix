myFlix Documentation

myFlix is a movie app is server-side built application that provides info about movies and its users. For this project I'm creating my own REST API and then implementating in REACT.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by its title including: title, description, genre, director, star actors, image URL, rating, featured. /movies/[Title] GET None A JSON object holding data about a single movie, containing title, director, and year. Example:
{
title: "The Godfather",
director: Francis Ford Coppola,
year: 1972
}
Return data about a genre, example: "Comedy". /genre/[genre] GET None A JSON object holding data about a genre. Example:
{
genre: "Comedy: Comedy is a genre of film in which the main emphasis is on humor. These films are designed to make the audience laugh through amusement and most often work by exaggerating characteristics for humorous effect.
Return data about a director (bio, birth year, death year by name. /directors/[director] GET None A JSON object holding data about a director.
director: Steven Spielberg,
{
directorID: 99, birthyear: 1946
}
Add a user /users POST A JSON object holding data about a new user. Example:
{
"user": "Jerry",
"email": "jerry@gmail.com",
"password": "jerry123",
"DOB": "November 1, 1970",
} 		A JSON object holding data about a new user with ID. Example:
{
"id": 2,
"user": "Jerry",
"email": "jerry@gmail.com",
"password": "jerry123",
"DOB": "January 1, 1970",
}
Allow user to update info /users/[username] PUT None A JSON object holding data about a new user. Example:
{
Name: "testuser1",
Password: "testuser123",
Mail: "testuser1@gmail.com"
"Birthday": "January 1st, 1971"
}
Add a movie to a user's favorites /users/[username]/movies/[movieID] POST None A message that a movie was added to the user's favorties.
Remove a movie by ID /users/[username]/movies/[movieID] DELETE None A text message that a movie has been removed from the user's favorites
Allow users to deregister /users/[username] DELETE None A message that a user has been removied
