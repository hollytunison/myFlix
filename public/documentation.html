myFlix Documentation

myFlix is a movie app is server-side built application that provides info about movies. For this project I'm creating my own REST API and then implementating REACT.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by its title /movies/[Title} GET None A JSON object holding data about a single movie, containing title, director, and year. Example:
{
title: "The Godfather",
director: Francis Ford Coppola,
year: 1972
}
Add a movie /movies POST A JSON object holding data about a movie to add. Example:
{
title: "Pulp Fiction",
directors: Quentin Tarantino,
year: 1994
} 		A JSON object holding data about a movie that was successfully added, including ID. Example:
{
title: "Pulp Fiction",
directors: Quentin Tarantino,
year: 1994
}
Remove a movie by its id /movies/[ID] DELETE None A JSON object holding data about the movie to add. Example:
{
title: "The Godfather",
director: Francis Ford Coppola,
year: 1972
}
Update a director of a movie /movies/[Title]/[Director] PUT None A text message indicating a director of a movie has been updated
